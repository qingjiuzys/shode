## Quick Reference Commands

### Building
```bash
# Build Go binaries
go build -o shode ./cmd/shode

# Build VS Code extension
cd vscode-shode
npm install
vsce package

# Build Docker images
docker build -t shode/registry:latest -f web-registry/backend/Dockerfile .
docker build -t shode/frontend:latest -f web-registry/frontend/Dockerfile .
```

### Testing
```bash
# Run all tests
go test ./...

# Run with coverage
go test -cover ./...

# Run performance benchmarks
go test -bench=. ./pkg/performance/...
```

### Git Tagging
```bash
# Create annotated tag
git tag -a v1.0.0 -m "Release v1.0.0: Production Ready"

# Push tag
git push origin v1.0.0

# Verify tag
git show v1.0.0
```

### Database Migration
```bash
# Run migrations
cd web-registry/backend
go run cmd/migrate/main.go up

# Verify schema
psql -h localhost -U shode -d shode_registry -f schema.sql
```

### Deployment
```bash
# Deploy backend
kubectl apply -f web-registry/backend/k8s/

# Deploy frontend
kubectl apply -f web-registry/frontend/k8s/

# Verify deployment
kubectl get pods -l app=shode-registry
```
