## ğŸš€ Deployment Steps

### 1. Pre-Deployment

```bash
# Run release script
./scripts/release.sh

# Review artifacts
ls -la bin/
ls -la vscode-shode/*.vsix
```

### 2. Create Git Tag

```bash
git tag -a v1.0.0 -m "Release v1.0.0: Production Ready"
git push origin v1.0.0
```

### 3. Deploy Services

```bash
# Deploy database migrations
kubectl apply -f web-registry/backend/migrations/

# Deploy backend
kubectl apply -f web-registry/backend/k8s/

# Deploy frontend
kubectl apply -f web-registry/frontend/k8s/

# Deploy Meilisearch
kubectl apply -f infrastructure/meilisearch/k8s/
```

### 4. Verify Deployment

```bash
# Check services
kubectl get pods -l app=shode-registry

# Run health checks
curl https://registry.shode.io/health
curl https://registry.shode.io/api/v1/packages
```

### 5. Publish Artifacts

```bash
# Publish VS Code extension
cd vscode-shode
npx vsce publish

# Create GitHub release
gh release create v1.0.0 \
  --title "v1.0.0: Production Ready" \
  --notes-file docs/v1.0.0-announcement.md
```
