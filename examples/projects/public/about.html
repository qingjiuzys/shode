About This Website

This website is built with Shode, a shell script platform that can run web servers.

Tech Stack

  • Backend: Shode (Shell Script)
  • Frontend: HTML + CSS
  • API: RESTful API powered by Shode
  • Hosting: Running on localhost

Features

  • ✅ Static file serving
  • ✅ RESTful API endpoints
  • ✅ Directory browsing
  • ✅ Gzip compression
  • ✅ Cache control
